#sakuravps

さくらVPSの初期設定を作成したChefのデータです。
WindowsCygwinの環境で、TUKUMEMO.COMが作成しました。

##内容

* rootパスワードの変更
* SSH接続の設定変更
* sudoの設定
* 一般ユーザーの追加
* システムのアップデート
* iptablesの設定
* WEBサーバーの設定
* WEBページの設定
* マルチドメインの設定
* PHPの設定
* MySQLの設定
* Postfix/Dovecot/Postfix Adminの設定
